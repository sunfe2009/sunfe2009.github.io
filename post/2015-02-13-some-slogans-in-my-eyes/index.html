我眼中的品牌Slogan

2015-02-13 2 min read # slogan # 品牌 # 广告 # 广告语

之前，一个叫做Viktor Hertz的歪果仁建立了一个叫做honest slogans的博客（honestslogans.com），专门用来吐槽品牌广告语，将自己对品牌的感受说出来，截止目前为止基本上人们叫得上名字的品牌都被他吐槽过了。

我看了这位歪果朋友的“作品”之后觉得很有意思，心想是不是应该来个中国版的，于是不久之后，我就将我认为品牌没有说出口的真相一一捅了出来。

我眼中这些品牌的Slogan应该是这样的：

和土豆合并后，优酷的前贴片广告真是越发嚣张了。

哈哈，不解释，大家都懂。

罗老师笑而不语~

双十一的时候感受最明显，剁手族心领神会。

你看我马叔叔都干了些什么啊，唉~

用过白加黑的朋友请给我一个姚明的表情~

NOKIA：Connecting Microsoft (他们最近被微软收购了)

原广告语是“为发烧而生”。

这里要插播一个著名的段子：

她来例假肚子疼，他坐在她旁边，看了她一眼，拿出手机玩游戏。她看在眼里，心里凉了半截。两分钟后，她实在坐不下去了，正准备离开，只见他默默地递过来他的小米手机说：拿去捂着。

很多人买kindle之前以为自己买了之后会看很多很多书的。

有多少人把麦记当作公共洗手间的？

长期使用盗版的童鞋是不是下意识捂住了钱包…

表示经常被带错地儿

和谷歌比起来，他们的Logo和每次的节日涂鸦简直丑到没朋友…

以上

就是某些我眼中的品牌slogan背后的真相，嗯，暂时先放这么多，感觉很快就会收到以上品牌的律师函了。PS：因为是去年写的，有些品牌的广告语可能已经变更，权当好玩，没做调整了。
